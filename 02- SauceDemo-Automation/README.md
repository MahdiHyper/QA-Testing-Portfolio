\# Selenium Automation Testing - SauceDemo



Automated testing framework for SauceDemo e-commerce website using Selenium WebDriver with Java and TestNG.



\## Project Overview



This project demonstrates automated testing using the Page Object Model (POM) design pattern. It covers the main user flows of the SauceDemo application including login, product browsing, and shopping cart functionality.



\## Technologies Used



\- \*\*Java\*\* - Programming language

\- \*\*Selenium WebDriver 4.27.0\*\* - Browser automation

\- \*\*TestNG 7.10.2\*\* - Testing framework

\- \*\*Maven\*\* - Build and dependency management

\- \*\*Edge WebDriver\*\* - Browser driver





\## Test Coverage



\*\*Login Tests (5)\*\*

\- Valid login

\- Empty username/password

\- Invalid credentials

\- Locked user



\*\*Product Tests (8)\*\*

\- Product page verification

\- Add/remove products

\- Sort functionality

\- Cart navigation



\*\*Cart Tests (5)\*\*

\- Cart page verification

\- Product count validation

\- Remove from cart

\- Continue shopping



\*\*Total: 18 automated tests\*\*
